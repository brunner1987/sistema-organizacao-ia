# Protocolos Técnicos

## Tabela de Formatação para ChatGPT

| Elemento | Estrutura | Exemplo |
|----------|-----------|---------|
| OBJETIVO PRINCIPAL | Resultado específico | "Criar email de vendas para produto X" |
| CONTEXTO COMPLETO | Background essencial | "Público-alvo: empresários 30-50 anos" |
| TAREFA DETALHADA | Passos específicos | "1. Escrever título 2. Desenvolver corpo" |
| FORMATO DE RESPOSTA | Estrutura esperada | "Lista com 5 tópicos principais" |
| RESTRIÇÕES | Limites importantes | "Evitar termos técnicos, tom informal" |

| Campo | Valores Possíveis | Função |
|-------|-------------------|--------|
| PROCESSO | input/processamento/saída | Estágio do conteúdo |
| PRIORIDADE | baixa/média/alta | Nível de urgência |
| TAGS | lista de tags | Classificação |
| CRIADO | data-hora | Timestamp criação |
| ATUALIZADO | data-hora | Timestamp modificação |
| STATUS | pendente/processado/finalizado | Estado atual |
