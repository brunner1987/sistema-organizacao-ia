# Sistema de Tags

| Tag | Função | Uso Recomendado |

| #processar | Precisa processamento com IA | Ideias iniciais, textos brutos |
| #refinado | Processamento completo | Conteúdo trabalhado com ChatGPT |
| #template | Modelo reutilizável | Estruturas para repetição |
